# Kullanıcı Profili ve İletişim Kuralları

- **Teknik Bilgi Seviyesi:** Kullanıcının teknik yazılım bilgisi bulunmamaktadır.
- **İletişim Dili:**
  - Açıklamalar mümkün olduğunca sade, anlaşılır ve teknik terimlerden arındırılmış (veya terimler açıklanarak) yapılmalıdır.
  - Öneriler sunulurken "neden" yapıldığı, kullanıcıya olan faydası üzerinden anlatılmalıdır.
  - Karmaşık teknik detaylar yerine, işlemin sonucuna ve kullanıcı deneyimine odaklanılmalıdır.
